# Wazuh SIEM

## Overview

This repository documents my hands-on learning journey with Wazuh during my cybersecurity internship and personal lab environment.

The objective is to understand how Wazuh works internally, deploy it from scratch, monitor endpoints, analyze security events, and develop practical SOC skills.

---

## Topics Covered

- SIEM Fundamentals
- Wazuh Architecture
- Manager
- Agent
- Indexer
- Dashboard
- Agentless Monitoring
- Deployment
- Rules & Decoders
- File Integrity Monitoring (FIM)
- Vulnerability Detection
- Active Response
- MITRE ATT&CK
- Security Configuration Assessment (SCA)
- Hands-on Labs
